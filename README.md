# 웨이브 몬스터 피하기 게임 (ver2.0)

## 게임 소개
- 플레이어(파란색 사각형)를 방향키로 조작하여 빨간 몬스터를 피하는 게임입니다.
- 몬스터는 화면 가장자리에서 등장해 플레이어를 향해 돌진합니다.
- 몬스터와 충돌하면 게임 오버, 몬스터가 화면 밖으로 나가면 피한 것으로 간주됩니다.
- 웨이브가 올라갈수록 몬스터 수와 속도가 증가합니다.

## 조작법
- 방향키(←, →, ↑, ↓)로 이동
- 게임 오버 시 'R' 키로 재시작
