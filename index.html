🎮 JavaScript vs Python 비교

  • 게임 루프 구조 동일 (requestAnimationFrame ↔ clock.tick)
  • 벡터 계산과 정규화 로직 동일
  • 충돌 검사 알고리즘 동일

  • Canvas API ↔ pygame 그래픽
  • 이벤트 리스너 ↔ pygame.event
